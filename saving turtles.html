HOW TO SAVE TURTLES


But there is still hope. We can save the turtles from extinction by practicing the following ways.


        

1. Reduce Plastic Usage
• Instead of using plastic for daily uses, we can use paper bags or cloth bags. When plastic usage
decreases, the disposal of plastic waste into the sea reduces, thus protecting the sea turtles.




        

2. Organize Beach Clean-Ups
• By cleaning up the beach periodically, we can remove exessive trash on the
beach site which can be destructive to a turtle's habitat.




        

3. Avoid Disturbing Turtle Nesting Sites
• This is because interfering a turtle's nesting site might frighten a female sea turtle while laying eggs.
This will lead the female turtle to escape back into the ocean instead of laying eggs on the beach site.




        

4. Do Not Support Turtle Poaching
• This can be done by not purchasing any souveniers, jewellery and furnitures made from body parts of a turtle.
Besides, we should also avoid purchasing turtle meat or eggs for consumption.




        

5. Skip A Straw
• Instead of using plastic straws, we can use reusable straws or paper straws. This is because the
disposal of plastic straws into the ocean reduces. At the same time, turtles would not get choked as
there will be no straws to get stuck in their nasal airways.
